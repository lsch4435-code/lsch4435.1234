# 오늘 뭐 먹지? 메뉴 추천기 🍙

JavaScript를 사용하여 사용자 입력을 기반으로 메뉴를 랜덤 추천해 주는 인터랙티브 웹 페이지입니다.  
사용자는 **메뉴 종류, 매운 정도, 가격대**를 선택하고, 버튼을 클릭하면 조건에 맞는 메뉴 중 하나가 화면에 동적으로 표시됩니다.

---

## 기능 소개

- 메뉴 종류 선택: 한식 / 중식 / 일식 / 분식 / 상관없음
- 매운 정도 선택: 안 매운 / 보통 / 매운 / 상관없음
- 가격대 선택: 저렴하게 / 보통 / 조금 사치하게 / 상관없음
- **추천 버튼 클릭 시**
  - 조건에 맞는 메뉴만 필터링
  - 그 중 하나를 `Math.random()`으로 랜덤 선택
  - 결과를 화면에 즉시 표시 (DOM 조작)

---

## 사용 기술

- HTML: 기본 구조 및 셀렉트 박스, 버튼, 결과 영역 구성
- CSS: 간단한 레이아웃 및 스타일링
- JavaScript:
  - `document.getElementById()`를 이용한 DOM 접근
  - `addEventListener()`로 클릭 이벤트 처리
  - 배열 `filter()`로 조건에 맞는 메뉴 필터링
  - `Math.random()` + `Math.floor()`로 랜덤 메뉴 선택
  - `textContent`로 결과를 동적으로 화면에 출력

---

## 실행 방법

1. 이 레포지토리를 로컬에 클론합니다.
2. `index.html` 파일을 브라우저(Chrome 등)에서 엽니다.
3. 옵션을 선택한 후 **"메뉴 추천 받기"** 버튼을 클릭합니다.

또는 GitHub Pages를 설정한 경우, 제공된 URL에 접속하여 바로 사용할 수 있습니다.
